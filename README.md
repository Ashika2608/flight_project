# Flight Booking System ✈️

Indha project oru Full-stack Flight Booking Application. Idhula backend matrum frontend rendu irukku.

## 🚀 Features
* **Search Flights:** Payanigal thangaLuku yetha flights-ai theda mudiyum.
* **Seat Selection:** Virumbiya seat-ai select seiyum vasadhi.
* **Booking History:** Munbu book seidha details-ai paarkalam.
* **Admin Dashboard:** Flights matrum bookings-ai manage seiya.

## 🛠️ Tech Stack
* **Frontend:** Flutter (Dart)
* **Backend:** Spring Boot (Java) - *[Unga backend vera edhavadhu na adhai mathunga]*
* **Database:** MySQL / H2 

## 📂 Project Structure
* `flight_frontend/`: Flutter mobile application code.
* `flight_backend/`: Spring Boot API code.

## ⚙️ How to Run
1. **Backend:** Project folder-kku poyi Maven/Gradle moolama run pannunga.
2. **Frontend:** `flutter run` command moolama emulator-la run pannunga.
